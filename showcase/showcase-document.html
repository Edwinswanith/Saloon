Point of Sale & Billing

A powerful, intuitive billing system designed for fast-paced salon operations. Create multi-item bills with services, packages, products, and memberships in seconds.

  • Multi-item checkout: services, packages, products, memberships & prepaid
  • Multiple payment methods: Cash, Card, UPI, Wallet
  • GST-compliant PDF invoice generation
  • Real-time inventory deduction on sale
  • Discount approval workflow with manager codes
  • Share invoices via Email, WhatsApp, or public link

Security & Access Control

Built with enterprise-level security to protect your business data across all branches and user roles.

JWT Authentication

Architecture

What Powers What

A detailed breakdown of which technology handles which part of the system.

⚙️

Frontend Layer React 18 + Vite 5 + Ant Design 6

Technology Used For Where in the App
React 18 Component-based UI rendering, SPA routing, state management Entire frontend — all 50+ pages/views are React components
Vite 5 Build tool, dev server, hot module replacement, production bundling Dev workflow & production build (with Terser to strip console.log)
Ant Design 6 UI component library — tables, forms, modals, date pickers, notifications Every page: QuickSale checkout, appointment forms, customer tables, report filters, settings panels
Recharts Data visualization — line charts, bar charts, pie charts, area charts Dashboard KPIs, revenue trends, payment distribution, staff performance, all report pages
Zustand Lightweight global state management Sidebar state, theme toggle, shared UI state across components
React Query Server state caching, auto-refetch, background sync All API calls — dashboard stats, customer lists, bill data, inventory, reports
Framer Motion Page transitions, mount/unmount animations, micro-interactions Page route transitions, modal animations, card hover effects
React Hook Form Form validation, controlled inputs, submit handling Customer creation, staff forms, service/package forms, settings forms
Day.js Date formatting, manipulation, range calculations Appointment calendar, report date filters, bill timestamps, membership expiry
React Icons Icon library for UI elements Sidebar navigation, action buttons, status indicators, KPI cards
🖥️

Backend Layer Python / Flask 3 + Gunicorn WSGI

Technology Used For Where in the App
Flask 3 Web framework — routing, request/response handling, Blueprints All 450+ API endpoints across 34 route modules (e.g. bill_routes, customer_routes, dashboard_routes)
Flask Blueprints Modular route organization, namespace isolation Each feature has its own Blueprint: /api/bills, /api/customers, /api/appointments, /api/staff, etc.
MongoEngine 0.27 MongoDB ODM — document models, queries, validation, indexing All 30+ data models: Bill, Customer, Staff, Appointment, Product, Service, etc.
PyJWT JWT token generation, validation, expiry management Login/auth system — generates tokens on login, validates on every @require_auth endpoint
bcrypt Password hashing and verification Staff, Manager, Owner password storage — hash on creation, verify on login
ReportLab PDF generation — professional invoice rendering Invoice PDF creation with GST breakdown, branch details, itemized billing
Gunicorn Production WSGI server with multi-worker threading Cloud Run deployment — 2 workers, 4 threads each, handles concurrent requests
flask-compress gzip response compression (70-80% size reduction) All API responses — especially large report data, customer lists, bill histories
Redis / In-Memory Response caching with TTL, falls back to dict-based cache Dashboard stats (5min TTL), report aggregations, frequently accessed data
MongoDB Aggregation Server-side data pipelines for analytics Staff leaderboard, revenue breakdown, payment distribution, client funnel, all report endpoints
🗃️

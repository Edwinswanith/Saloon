# Bizzzup Quick Sale

A responsive React application replicating the Bizzzup "Quick Sale" billing screen UI.

## Features

- Fixed purple sidebar with navigation menu
- Responsive "New Booking" card with all billing components
- Clean, light theme with subtle borders and white space
- Mobile-friendly responsive layout

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

The application will be available at `http://localhost:5173`

### Build

```bash
npm run build
```

## Project Structure

```
src/
  ├── components/
  │   ├── Sidebar.jsx       # Navigation sidebar component
  │   ├── Sidebar.css
  │   ├── QuickSale.jsx     # Main billing form component
  │   └── QuickSale.css
  ├── App.jsx               # Main app component
  ├── App.css
  ├── main.jsx              # Entry point
  └── index.css             # Global styles
```

## Technologies Used

- React 18
- Vite
- CSS3 (no external UI libraries)
